= Настройка подписания

Для карточки вида _Документ_ можно настроить подписание полей xref:card-kinds/document/sign-card.adoc[карточки] и xref:card-kinds/document/sign-operation.adoc[операций] редактирования, выполняемых с карточкой. Наложение электронной подписи возможно для _Основных_ и _Дополнительных_ файлов, вложенных в карточку.

Вид подписи, которая формируется при подписании карточек и файлов, определяется настройками, выполненными администратором {dv} в справочнике _Системные настройки_.
