.Чтобы назначить пользователям права доступа к ссылке:
. Откройте _Справочник ссылок_.
. Щелкните правой кнопкой мыши по объекту, права доступа к которому необходимо назначить.
. Чтобы отобразить окно назначения прав доступа, выберите из контекстного меню команду _Безопасность_.
. Настройте права доступа с помощью стандартного диалога управления правами.
+
Ограничения прав, наложенные на первую часть двусторонней ссылки -- прямую ссылку, автоматически будут применяться и ко второй части пары -- обратной ссылке. Однако в окне, открывающемся при выборе из контекстного ссылки команды _Безопасность_, это отражено не будет.
