= Управление ролями

В _Справочнике сотрудников_ существует возможность объединения сотрудников по ролям.

Для управления ролями предназначена вкладка xref:staff/directory.adoc#roles-tab["Роли"], на которой выводятся роли и список сотрудников и подразделений в них.

В созданную роль можно добавлять сотрудников или удалять сотрудников из роли по аналогии с организациями и подразделениями.
