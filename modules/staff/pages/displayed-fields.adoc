= Отображаемые поля организаций и подразделений

По умолчанию на правой панели справочника отображаются название, код и телефон организации или подразделения. Настройки могут быть переопределены на любом уровне. Настройки отображения распространяются на все подчиненные подразделения.

[WARNING]
====
Настройку отображаемых полей сотрудников, подразделений и групп могут производить только администраторы {dv}.
====

.Чтобы настроить отображаемые поля организаций или подразделений:
. Выделите нужный объект в дереве на левой панели справочника, затем нажмите на кнопку image:buttons/dept-displayed-fields.png[Отображаемые поля подразделений] на ленте справочника.
+
include::partial$excerpts.adoc[tags=groups]
+
. Будет открыто окно _Отображаемые поля подразделений_.
+
include::partial$excerpts.adoc[tags=up-down]
+
.Отображаемые поля подразделений
image::staff-dept-fields.png[Отображаемые поля подразделений]
+
. Чтобы добавить новое поле в список отображаемых в справочнике, нажмите кнопку *Добавить* image:buttons/plus-green.png[Зелёный плюс].
+
.Выбор поля для отображения
image::staff-select-displayed-field.png[Выбор поля для отображения]
+
. В окне _Поля для отображения_ Нажмите на кнопку image:buttons/three-dots.png[Три точки] справа от поля.
+
Появится список доступных полей, которые используются для отображения информации справочника.
+
.Список полей, доступных для выбора при настройке отображаемых полей подразделений
image::staff-selectable-fields.png[Список полей, доступных для выбора при настройке отображаемых полей подразделений]
+
. Выберите поле, которое должно отображаться в справочнике.
. Чтобы в соответствующей полю колонке справочника отображался только первый символ из выбранного поля, установите флаг `*Отображать только первый символ*`.
. Чтобы вернуться к стандартному набору отображаемых полей, в окне _Отображаемые поля подразделений_ удалите все явно указанные поля.
