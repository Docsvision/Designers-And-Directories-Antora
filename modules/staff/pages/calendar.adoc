= Календарь

По умолчанию для организации используется календарь, назначенный администратором системы Бизнес-календарем по умолчанию в справочнике _Системные настройки_. При необходимости для каждой организации может быть назначен собственный бизнес-календарь, который будет иметь приоритет над системным календарём.

.Чтобы определить рабочий календарь для организации:
include::partial$excerpts.adoc[tags=open-window]
. Перейдите на вкладку _Настройки делопроизводства_.
. В области _Календарь_ в поле _Рабочий календарь_ выберите календарь, который будет использоваться для подразделений и сотрудников организации при вычислении сроков исполнения заданий.
