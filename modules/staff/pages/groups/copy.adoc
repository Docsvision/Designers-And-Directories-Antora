= Скопировать группу

Группа может быть скопирована в другой узел иерархии групп.

. Выделите группу, которую требуется скопировать.
. Правой клавишей мыши вызовите контекстное меню, затем выберите команду _Копировать..._ или выделите нужную группу и перетащите её на нужную позицию.
+
include::partial$excerpts.adoc[tags=group-name]
+
Будет открыт на выбор _Справочник сотрудников_.
+
.Вид справочника сотрудников при выборе целевой группы
image::staff-dir-select-group.png[Вид справочника сотрудников при выборе целевой группы]
+
. Выберите целевую группу, затем нажмите на кнопку image:buttons/check.png[Зелёная галочка].
