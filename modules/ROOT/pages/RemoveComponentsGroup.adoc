== Удаление группы компонентов

. [.ph .cmd]#Откройте [.dfn .term]_Справочник компонентов_.#
. [.ph .cmd]#Выберите группу компонентов в списке групп.#
. [.ph .cmd]#На панели инструментов нажмите кнопку image:img/Buttons/removeComponentsGroup.png[image]. Группа компонентов будет удалена из списка групп.#
. [.ph .cmd]#На панели инструментов нажмите кнопку image:img/Buttons/saveComponentsDirectory.png[image]. [.dfn .term]_Справочник компонентов_ будет сохранен. Компоненты, которые были загружены в удаленную группу, перестанут распространяться.#
