= Performing

== Назначение

Группа элементов отображает панель _Ход исполнения_.

.Группа включает несколько сгруппированных элементов:
* `_Percent completed_`.
* группа `_Actual performing date_`.
* `_Completed user_`.
* `_Текущие исполнители_`.
* группа `_Comments_`.
